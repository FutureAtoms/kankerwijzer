The Problem

Trusted cancer information is
scattered. People use ChatGPT instead.

“I just asked ChatGPT because I didn’t know which IKNL website to check.”

Patients and families turn to general AI because it's fast and easy. But generic AI sounds confident while being wrong, incomplete, or uncited.

✗ Answers are fast, but not reliable

Generic AI can hallucinate medical facts with no source trail.

✗ Trusted sources exist, but are hard to find

Information is spread across 7+ IKNL-related sites, APIs, reports, and guidelines.

✗ Context is lost when sources are separated

Medical information loses meaning when disconnected from its provenance.

What “Medical-Grade” Means

Not a diagnostic system.
A medically careful information system.

✓

Source-locked evidence

Only the 7 approved source families enter the evidence pool. No external web results.

✓

Full provenance on every chunk

URL, document ID, chunk ID, page number, section, checksum, fetch time.

✓

Structured APIs before embeddings

When exact numbers matter, the system calls NKR / Cancer Atlas APIs first.

✓

Abstention on weak evidence

If retrieval scores fall below a calibrated threshold, the system refuses instead of guessing.

✓

Conflict flagging

Contradictory sources are surfaced, not smoothed over.

✓

Red-flag symptom routing

Emergency symptoms trigger “Bel 112” routing. Suicidal ideation routes to 113.

✓

Server-side citation assembly

The LLM uses [SRC-N] labels. The server maps them to Provenance objects. The model never authors URLs.

✓

Sensitive data stays client-side

Lastmeter distress scores never leave the browser. Only domain keywords are sent.
